# NLP - Natural Language Processing

## Description
Ce repository contient les travaux pratiques réalisés dans le cadre du cours de Natural Language Processing (NLP). Les TDs explorent différentes techniques de représentation textuelle, du prétraitement de base aux embeddings sophistiqués.

##  TD1 - Représentations de Mots et de Phrases

Comprendre et implémenter différentes méthodes de vectorisation de texte pour permettre aux algorithmes de machine learning de traiter du texte.

##  TD2 - Pipeline de Prétraitement Avancé

Construire un pipeline complet de prétraitement pour transformer des textes bruts en représentations exploitables pour la modélisation de sujets.
